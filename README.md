## Hardware

- **Keyboard**: Sofle v2
- **Firmware**: [ZMK](https://zmk.dev/)
- **Displays**: Nice OLED screens ([zmk-nice-oled](https://github.com/mctechnology17/zmk-nice-oled))
- **LEDs**: RGB LED support via zmk-nice-oled
